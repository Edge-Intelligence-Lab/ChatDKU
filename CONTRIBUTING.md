# Contributing to ChatDKU

Thanks for your interest in contributing!

## Quick Start

1. Fork the repo and create a feature branch.
2. Copy `.env.example` to `.env` and configure your local settings.
3. Run the stack using Docker or your preferred local setup.

## Development

- Python: `pip install -e chatdku`
- Frontend: `npm install` in `chatdku/chatdku/frontend/ChatDKU-web`

## Tests

- Python: `pytest`
- Frontend: `npm test`

## Code Style

- Python formatting: `black`
- Python linting: `flake8`

## Pull Requests

- Keep changes focused and include a clear description.
- Add or update tests where appropriate.
- Avoid committing secrets or private data.

## Reporting Issues

Please open an issue with reproduction steps and expected behavior.
